# [PC CLIM](https://www.pc-clim.fr)

### Contexte
Dans le cadre de l'agrandissement de son équipe, l'entreprise PC CLIM souhaite avoir
un site vitrine qui permet de faire un résumé de l'activité, présenter les réalisations
de l'entreprise et posséder un visuel des membres de l'équipe (nom, prénom, photo).

### Stack Technique
- PHP
- Symfony
- TailwindCSS
- Javascript
- MySQL
- Linux
- OVH
- IONOS
